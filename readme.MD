## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/sanjanastyles/Titan.git
   cd titan
   ```
2. Install dependencies:
   ```bash
   npm install
   ```

## Usage

- To start the server:

```bash
npm start
```

- To start the server in Development:

```bash
npm run dev
```
